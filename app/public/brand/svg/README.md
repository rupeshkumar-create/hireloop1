# Hireloop — Custom Asset Pack

Charcoal + electric-lime · Geist · drawn on a 24 grid, 1.75 stroke, round caps/joins.
Signature: every glyph carries one lime "live-node" (#B9F84C).

## Contents
- logo/      mark (lime / charcoal / white), app-icon, wordmark lockups
- icons/     61 UI icons (currentColor line + lime node)
- arrows/    12 arrows & loop returns

## Usage
Icons/arrows use stroke="currentColor" — set color to theme them.
The lime node is hard-coded #B9F84C so the accent survives recolouring.
